# CERTEZAS CONFIRMADAS — Proyecto EITEC

Este documento recopila las certezas estratégicas de negocio, marca y diseño confirmadas por Marcelo (presidente de EITEC) y por el análisis del negocio actual.

## [C6] EITAR es marca recontra posicionada en el mercado de reposición
**Área:** SEO / Marca  
**Contexto:** Los clientes buscan activamente "repuestos EITAR", "válvulas EITAR", "termocupla EITAR". El SEO debe capitalizar esta búsqueda orgánica.  
**Impacto:** Afecta H1, meta-etiquetas, copy de categorías, estructura de URLs y posibles landing pages de búsqueda por marca histórica.

## [C7] El canal de venta es B2B: mayoristas, ferreteros sanitarios, casas de sanitarios
**Área:** Negocio / UX  
**Contexto:** No hay intención confirmada de venta directa al público final. El sitio debe generar leads mayoristas, no ventas minoristas.  
**Impacto:** Afecta CTAs, formularios, iconografía (carrito de compras), copy de contacto y la arquitectura de información del sitio.

## [C8] Un tercero compró el dominio/marca EITAR
**Área:** Legal / Riesgo / SEO  
**Contexto:** Estado actual desconocido (¿caído? ¿vigente?). Esto representa un riesgo de competencia y una oportunidad de recuperación de dominio.  
**Impacto:** Afecta decisiones de redirecciones 301, autoridad de dominio, registro defensivo de dominios y estrategia de marca comparativa.

## [C9] El carrito de compras en el Header contradice el modelo de negocio real
**Área:** UX / Negocio  
**Contexto:** El ícono de carrito en el header genera confusión B2B. Debe eliminarse para evitar que los visitantes interpreten el sitio como e-commerce minorista.  
**Impacto:** Afecta `templates/partials/header.html`, `static/js/main.js` y posiblemente la ruta `/carrito`.

## [C10] Dominio `eitar.com.ar` registrado por tercero hasta 2027
**Área:** Legal / SEO / Dominio  
**Contexto:** El dominio `eitar.com.ar` está registrado a nombre de GUAZZINI BRUNO DARIO, con fecha de alta 26/02/2021 y vencimiento 26/02/2027. No está disponible para registro.  
**Impacto:** Define que la recuperación del dominio requiere negociación con el titular o espera al vencimiento (con riesgo de renovación por parte del titular). No se puede registrar el dominio exacto.

## [C11] Dominio `eitar.ar` está disponible para registro
**Área:** Legal / SEO / Dominio  
**Contexto:** El dominio de primer nivel bajo `.ar` (`eitar.ar`) se encuentra disponible. Es más corto, moderno y fácil de recordar que `eitar.com.ar`.  
**Impacto:** Oportunidad de registrar defensivamente el dominio exacto de marca histórica y redirigirlo a `eitec.coop.ar`.
